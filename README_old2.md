**MonWatch-UI**  
MonWatch-UI is a tool for browsing and previewing Himawari satellite data with CRS georeferencing.

**Features**  
• Automatic Python setup via run.bat.  
• Interactive UI with drag-and-drop support.  
• Manual folder navigation (date → time → band).  
• CRS georeferencing support.

**Requirements**  
• Windows 10/11.  
• Internet connection.

**Installation**  
1. Clone or download the repo.  
2. Run run.bat.  
3. UI launches automatically.

**Usage**  
• Drag-and-drop AWS Himawari files.  
• Or use Open Folder to navigate date/time/band.

**Data Source**  
AWS NOAA Himawari imagery with CRS support.
